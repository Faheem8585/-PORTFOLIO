## Overview

An intelligent assistant suite for computational fluid dynamics (CFD) professionals, combining conversational AI with technical documentation support.

## Key Features

- **Dual Mode System**: CFD GPT and OpenFOAM GPT
- **File Upload Support**: PDF, images with OCR, Word documents
- **LaTeX Rendering**: Mathematical equation visualization
- **Modern UI**: Professional Streamlit interface
- **Document Processing**: Multi-format document analysis

## Technical Components

- **AI Integration**: OpenAI API for intelligent responses
- **OCR**: Optical character recognition for images
- **Document Parser**: Multi-format document handling
- **UI Framework**: Streamlit with custom styling

## Use Cases

- CFD problem-solving assistance
- OpenFOAM configuration help
- Technical documentation analysis
- Educational support for CFD learners
