## Overview

Real-time facial emotion recognition system using deep learning to detect and classify human emotions from webcam feeds or static images.

## Features

- **7 Emotion Classes**: Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise
- **Real-time Detection**: Live webcam emotion detection
- **CNN Architecture**: Custom deep learning model
- **Interactive UI**: Streamlit web application

## Technical Implementation

- Dataset: FER2013 (35,000+ facial images)
- Model: Convolutional Neural Network (CNN)
- Training: 50 epochs with data augmentation
- Deployment: Streamlit for easy access

## Applications

- Human-computer interaction
- Customer sentiment analysis
- Mental health monitoring
- Educational platforms
