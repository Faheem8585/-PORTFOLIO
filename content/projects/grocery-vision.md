## Overview

An intelligent computer vision system designed for automated grocery item detection, suitable for retail automation and smart checkout systems.

## Key Features

- **Real-time Detection**: Fast and accurate grocery item recognition
- **Cloud Deployment**: Live demo on Streamlit Community Cloud
- **Scalable Architecture**: Handles multiple items simultaneously
- **User-Friendly Interface**: Interactive web application

## Technical Implementation

- **Object Detection**: YOLOv8-based detection pipeline
- **Image Processing**: OpenCV for preprocessing
- **Web Interface**: Streamlit for deployment
- **Model Training**: Custom dataset training for grocery items

## Applications

- Smart retail checkout systems
- Inventory management automation
- Self-service kiosks
- Retail analytics

## Deployment

Successfully deployed to Streamlit Cloud, demonstrating production-ready computer vision capabilities in a cloud environment.
