## Bachelor's Thesis Project

A comprehensive mobile donation platform that bridges the gap between charitable recipients and donors through technology.

## Features

- **Dual User System**: Separate interfaces for recipients and donors
- **Admin Review**: Verification system for donation requests
- **Real-time Notifications**: Push notifications for donation updates
- **Secure Payments**: Integrated card payment processing
- **Progress Tracking**: Visual donation goal tracking
- **Firebase Backend**: Scalable cloud infrastructure

## Technology Stack

- **Frontend**: Flutter/Dart for cross-platform mobile development
- **Backend**: Firebase (Firestore, Authentication, Cloud Functions)
- **Payments**: Secure card payment gateway integration
- **Notifications**: Firebase Cloud Messaging (FCM)

## Impact

Successfully deployed as a Bachelor of Science in Computer Science thesis project, demonstrating full-stack mobile development capabilities and addressing real-world social impact needs.
